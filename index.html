Preguntas Frecuentes

Dentro de nuestros servicios y/o trabajos pueden surgir una gran cantidad de incógnitas que muchas veces no quedan muy claras.

  • ¿A que se refiere con dejar una documentación por un proyecto, trabajo o servicio?.

    Cada proyecto, trabajo web o servicios web que es solicitado por el cliente debe ser documentado y firmado por parte del cliente y por parte del jefe de proyecto.

  • ¿Que son las actualizaciones de un proyecto, trabajo o servicio?.

    Cuando el proyecto es entregado y finalizado (firmado en el acta de cierre). El cliente puede sugerir cambios y/o mejoras contínuas de algunas funciones o módulos, aquellos cambios se van a un informe de mejoras contínuas, esto también significa un monto aparte.

  • ¿Cuanto es el tiempo de demora de un proyecto, trabajo o servicio?

    Depende del proyecto de cada cliente, de las funciones solicitadas, del nivel de exigencia, etc.

  • ¿Cual es el prosupuesto del proyecto, trabajo o servicio?.

    Depende de los proyectos que se van solicitando, el tiempo, requerimientos, la cantidad de funciones, las tecnologías a usar y el nivel de complejidad del proyecto.
